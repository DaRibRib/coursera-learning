Probably lecture 8 because the last thing I did was lecture 7 - HTML character entity references

So in HTML, you weirdly can't use some characters you could usually use on your keyboard (like "greater than", "less than" and "and" symbols) So you are forced to use codes in order to denote these symbols. (Also used for stuff which isn't on the reglar everyday keyboard, like the © symbol.)
Here are the CHARACTER ENTITY REFERENCES you need to know for the more common stuff:
  • < -- & l t ;
  • > -- & g t ;
  • & -- & a m p ;
  • © -- & c o p y ;
Note: If you put these characters together, it would create the desired symbol, so I spaced them out so that didn't happen. When actually coding, remember to put these together.
Also found out about character reference to wrap text together (essentially meaning that the words will always stay together even if you shrink the page). Could use this to link words with special meaning together. In order to do so, take away the space between the words and add "& n b s p ;".
